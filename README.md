### 简介

本项目是一个仿写MAC网页版QQ项目。

### 项目使用技术栈

本项目使用到的技术栈如下：
* React全家桶
* TypeScript

## 加入项目

如果想加入本项目的开发，需将项目fork到自己的仓库下，功能开发完成后，提PR即可。
### 项目说明

当前分支为eslint版本，项目使用eslint + prettier统一风格，尚有未处理bug

启动项目

```
yarn install
```

```
yarn serve
```

项目打包

```
yarn build
```
